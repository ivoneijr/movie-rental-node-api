Projeto criado para primeira fase do processo seletivo da empresa 4all.

Para rodar o projeto execute os seguintes comandos: 
    npm install
    node app.js

Por padrão a aplicação irá subir no endereço http://localhost:9999/

Documentação da API está em http://localhost:9999/ ou http://localhost:9999/index.html
